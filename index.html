Lecture du moteur

1. Probabilité brute Chaque cote est transformée avec 1 divisé par la cote.
2. Marge du marché La somme des probabilités dépasse souvent 100 %, c'est la marge incluse dans les cotes.
3. Probabilité normalisée Le moteur retire l'effet de marge pour comparer proprement les profils 1 / N / 2.
4. Matchs similaires Il cherche dans la base historique les anciens matchs dont les probabilités ressemblent le plus.
